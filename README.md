```bash
cd <repo-name>
open index.html
```

Or drag `index.html` into any browser.

## Deploying with GitHub Pages

1. Push both `index.html` and `style.css` to your repository
2. Go to **Settings → Pages**
3. Under **Source**, select `Deploy from a branch`
4. Choose `main` branch → `/ (root)` → click **Save**
5. Your site will be live at:

```
https://namish-yadav.github.io/<repo-name>/
```

## Tools Covered

- **AltStore** — Sideload apps using your Apple ID
- **Sideloadly** — Windows/macOS sideloading tool
- **Xcode** — Apple's official IDE, supports direct device installs
- **Scarlet** — No-jailbreak iOS app installer
